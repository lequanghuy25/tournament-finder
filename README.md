# FIDE Tournament Finder

Web app tim kiem cac giai dau FIDE-rated de lap ke hoach thi dau.

[![Deploy to Render](https://render.com/images/deploy-to-render-button.svg)](https://render.com/deploy?repo=https://github.com/lequanghuy25/tournament-finder)

## Chay local

```powershell
npm install
npm start
```

Mo:

```text
http://localhost:4173
```

## Kiem tra ket noi FIDE

```text
/api/diagnostics
```

Neu server online tra `ok: true`, app co the lay du lieu that tu FIDE.
